# Student Mental Health & Burnout Dashboard

## Project Overview
This Power BI dashboard analyzes student mental health and burnout using interactive visualizations. It provides insights into stress levels, burnout factors, academic pressure, and lifestyle habits to support data-driven decision-making.

## Objectives
- Analyze student mental health trends
- Identify factors contributing to burnout
- Monitor stress and academic pressure
- Create interactive dashboards for better insights

## Tools Used
- Power BI
- Microsoft Excel
- DAX
- Power Query

## Dataset
The dataset includes student information such as:
- Age
- Gender
- Academic Pressure
- Study Hours
- Sleep Duration
- Burnout Level
- Stress Level
- Mental Health Status

## Dashboard Features
- KPI Cards
- Burnout Analysis
- Stress Level Analysis
- Academic Pressure Analysis
- Gender-wise Comparison
- Interactive Filters and Slicers

- Data Cleaning

## Conclusion
This dashboard helps identify the major factors affecting student mental health and burnout, enabling educators and decision-makers to understand trends through interactive visualizations.
